CSS Box Model

Content-Box Model

This box uses the content-box model. Padding, border, and margin are added outside the content box.

Border-Box Model

This box uses the border-box model. Padding and border are included within the width and height of the box.

Padding Example

This box demonstrates padding. Padding adds space inside the border.

Border Example

This box demonstrates borders. Borders add space around the content and padding.

Margin Example

This box demonstrates margins. Margins add space outside the border.
